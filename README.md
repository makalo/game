    pip install tensorflow
    pip install pygame
    pip install opencv_python-3.2.0-cp35-none-win_amd64.whl
    
    For Linux
    Both python2.7 and python3.5 work well
    pip install tensorflow
    pip install pygame
    apt-get install opencv
